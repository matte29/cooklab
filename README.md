# CookLab

#

This is the backend for a recipe / cookbook website.

Tech:

-   Nest.Js
-   Typescript
-   Postgre SQL
-   TypeORM
-   JWT
-   Passport
-   Jest

Set up a local postgres sql database and change the username and password in 'src/config'typeorm.config.ts' to the username and password for your server.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
